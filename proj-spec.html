Heist at the Museum - Game Project

Project Description

Game Type: Capture the Flag Meets Night at the Museum!

Description: A 3v1 game where thieves attempt to steal artifacts from a museum and return them to their countries of origin while the security guard tries to stop them (at least until their shift ends).

Win Conditions:

  • Thieves win by escaping the museum with the artifact they're targeting
  • Security guard wins if the artifact is still in its place by the end of their shift

Unique Aspects:

  • Unique storyline: returning stolen artifacts to their countries of origin
  • Teamwork among thieves to evade detection.
  • Security guard with unique, possibly supernatural abilities.

Game Features

Must Have

  • Museum with main room and two smaller rooms, all filled with artifacts
  • NPCs that roam around the museum, thieves can blend in with them to evade the security guard
  • Security guard has different items to help them catch the thieves: taser gun (ranged), baton (melee), etc.
  • Security guard has special abilities (e.g. see thief location).
  • Thieves can pick up artifact they're targeting

Would Be Really Nice

  • Thieves can change appearance.
  • Target Artifact is fragile and can be broken (complicates win conditions)
  • Storyline continuity, where each round, the security guard is essentially taking over for the security guard from the previous round, so now they're the ones who have to deal with the thieves until the next shift
  • Thieves have unique roles, each with their own special abilities/items.

Cool But Only If Ahead of Schedule

  • Alive artifacts that can move on their own
  • Character customization.
  • Artifact fragility applied to all items.
  • Thieves have in-game communication system.
